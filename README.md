## Project Overview
This project is a Cyber Security Chatbot developed in C# using WPF. The chatbot educates users about cybersecurity topics while providing an interactive graphical user interface.

## Features

### Part 1
- Graphical User Interface (GUI)
- Cybersecurity keyword responses
- Personalized greeting
- Input validation

### Part 2
- Memory for previous topics
- Sentiment detection
- Follow-up questions
- Cybersecurity quiz

### Part 3
- Task Assistant
  - Add tasks
  - View tasks
  - Delete tasks
- Task storage using JSON
- Activity Logger
- Improved chatbot integration

## Technologies Used
- C#
- .NET
- WPF
- Visual Studio 2022
- JSON

## How to Run
1. Open the solution in Visual Studio.
2. Build the project.
3. Run the application.
4. Interact with the chatbot through the graphical interface.
