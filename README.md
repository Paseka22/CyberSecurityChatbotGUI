# Cyber Security Assistant

A smart WPF cybersecurity chatbot developed in C# and .NET 8 that helps users learn about cybersecurity awareness and online safety.

## Features
- GUI chatbot interface using WPF
- Keyword recognition for cybersecurity topics
- Random chatbot responses
- Sentiment detection
- Memory and personalised replies
- Follow-up conversation flow
- ASCII art header
- Voice greeting support

## Technologies Used
- C#
- WPF
- .NET 8
- Visual Studio 2022

## How to Run
1. Open the solution in Visual Studio 2022
2. Build and run the project
3. Start chatting with the Cyber Security Assistant

## Project Files
- MainWindow.xaml
- MainWindow.xaml.cs
- Chatbot.cs
- KeywordResponder.cs
- SentimentDetector.cs
- MemoryStore.cs
